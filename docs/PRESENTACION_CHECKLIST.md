# Checklist de Presentacion

Usa esta secuencia para la exposicion en clase:

1. Presentar el problema real que resuelve PanamaAlert.
2. Mostrar el modelo relacional y explicar por que cumple 3FN.
3. Enseñar las vistas y procedimientos almacenados.
4. Hacer un CRUD corto desde la app o Postman.
5. Mostrar el ETL ejecutado y su salida.
6. Mostrar el tablero o la conexion a BI.
7. Enseñar las pruebas de Postman y JMeter.
8. Cerrar con seguridad, robustez y mejoras implementadas.

## Evidencias que conviene tener listas

- mapa con incidentes
- panel admin
- scripts SQL
- Postman
- JMeter o reporte exportado
- documento PDF final
