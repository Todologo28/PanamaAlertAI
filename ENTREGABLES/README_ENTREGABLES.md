# Entregables PanamaAlert

Esta carpeta contiene la documentacion formal del proyecto PanamaAlert para la entrega final de Base de Datos II.

## Archivos incluidos

- `01_CASO_DE_USO_Y_CUMPLIMIENTO.md`: explicacion detallada del caso de uso y como se implementaron los requisitos.
- `02_DOCUMENTACION_HERRAMIENTAS.md`: documentacion de las herramientas utilizadas.
- `03_MANUAL_PASO_A_PASO.md`: manual operativo para instalacion, ejecucion, pruebas y uso.
- `04_LECCIONES_APRENDIDAS.md`: reflexion tecnica y aprendizajes del proyecto.
- `05_GLOSARIO_TERMINOS.md`: glosario de terminos tecnicos usados en la solucion.

## Evidencias que se deben anexar

- Captura del diagrama entidad-relacion.
- Capturas de Power BI conectado por ODBC a MariaDB.
- Capturas de Postman ejecutando la coleccion.
- Capturas de JMeter con `Summary Report`, `Aggregate Report` y resultados exitosos.
- Capturas de la app web: login, mapa publico, mapa autenticado, perfil, dashboard y asistente.

## Recomendacion de entrega

Convertir estos archivos a PDF o copiarlos a un documento Word final, agregando capturas debajo de cada seccion correspondiente.
